# Simplify Backend

This is the backend of the bookstore project for "Rapid Application Development Module of GDSE67".

## Installation

All the necessary dependencies are included in the setup.bat file. Open command console and run

```bash
setup.bat
```
command to run the setup file. Or simply double click it.

To be used with sql database first the MySql server needs to be installed.

"db.sql"  has the necessary queries for the database installation. Every other table will be created upon running the project.

## Run Locally

Clone the project

```bash
git clone https://github.com/Dreaca/react_assignment_backend.git
```

Go to the project directory

```bash
cd my-project
```

Install dependencies

- Just run the setup.bat file

Start the server

- The package.json contains a script named "nodemon server.ts" as "start". Simply configure a npm configuration to run "start"

## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express
